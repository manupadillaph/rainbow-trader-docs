    same_hypers_across_folds

    • Each experiment creates its own Optuna study.
    • All studies start with the same sampler seed: 2026.
    • While different folds have the same trial history, the same optuna_trial_number uses the same hyperparameters.
    • Verified first divergence point by model:
    Model Same Hyperparameters Across Folds Until First Divergence
    mlp Trial 7 Trial 8
    tcn Trial 10 Trial 11
    vit Trial 7 Trial 8
    axial_vit Trial 7 Trial 8
    logistic Trial 9 Trial 10
    • This means early fold-to-fold differences are not explained by "Optuna tried different settings".
    • They are explained by the same settings behaving differently on different fold data, then TPE diverging later because the observed outcomes are no longer the same.
    Model Same Trial Number Same Hyperparameters Fold Outcome A Fold Outcome B
    logistic 9 Yes F2: COMPLETED, objective=0.2419 F1: PRUNED by distribution_drift
