    Methodology

    This report evaluates whether trained models produce statistically superior objective_value distributions compared to random baselines.

    What is being compared?

    Each trained model (e.g. mlp, tcn, vit) runs multiple trials during training. Each trial produces an objective_value score. We pool the objective_value scores from completed trials and compare them against the scores from random baseline models using a statistical test.

    Two levels of analysis

    1. Grouped analysis (main sections below) — experiments are pooled by combinations of {Run, Tier, Model, Fold} dimensions. For example, "By Model" pools ALL trials from every mlp experiment into one group, then tests that group against each random type. This answers: "Does this model type beat randoms overall?"
    2. Raw pairwise analysis (Appendix A) — each individual experiment is compared 1-to-1 against each random baseline of the same fold. This answers: "Does this specific experiment beat this specific random?"

    Random baselines

    Three random baseline types exist, only in Tier A: RLA (random_l), RBO (random_b), and RMK (random_markov). Every real model is always compared against all three separately — randoms are never pooled together.

    When fold is part of the grouping dimensions, randoms are filtered to the matching fold. When fold is NOT in the grouping, all random trials across folds are pooled together.

    Statistical test

    • Mann-Whitney U two-sided test (non-parametric, compares rank distributions)
    • Holm-Sidak correction for multiple testing (applied independently within each section)
    • Bootstrap 95% CI for median difference (1000 iterations, seed 2026)

    PASS criteria (all three must hold)

    1. median(real) > median(random) — the trained model must outperform
    2. Holm-Sidak corrected p < 0.05 — statistically significant
    3. rank-biserial > 0.3 — medium or larger effect size

    How to read the tables

    Column Meaning
    Group The combination of dimension values that defines this group
    vs Random Which random baseline type is being compared against
    n_exp Number of real experiments contributing ≥1 completed trial
    n_real / n_rand Number of pooled trial scores on each side
    Med Real / Med Rand Median objective_value for each side
    Diff Median difference (Med Real − Med Rand); positive = real is better
    p Raw Mann-Whitney p-value
    p_corr Holm-Sidak corrected p-value (accounts for multiple comparisons)
    Effect Rank-biserial effect size (−1 to +1; >0.3 = medium effect)
    CI 95% Bootstrap 95% confidence interval for the median difference
    Status PASS if all 3 criteria met, FAIL otherwise, SKIP if no data
