    What Was Learned

    • logistic stays out because every Tier A fold already has >= 10 completed.
    • mlp, tcn, vit, and axial_vit stay in because at least one fold remains below the 10 completed rule.
    • Early fold-to-fold differences are not explained by different sampled hyperparameters:
    • mlp: same through trial 7
    • tcn: same through trial 10
    • vit: same through trial 7
    • axial_vit: same through trial 7
    • MLP-F4, TCN-F1, TCN-F2, and AXV-F1 still contain untested transfer hypotheses from sibling folds with completed trials.
    • AXV-F2 gives the first completed axial_vit survivors in Tier A and proves a viable region exists.

    Rescue Decision

    • Keep Tier A only.
    • Keep trainable models only.
    • Keep scoring, labels, simulation, and prune logic unchanged.
    • Use one rescue experiment per model/fold.
    • Use n_trials = 20.
    • Use:
    • exact replay seeds
    • neighbor seeds
    • narrowed TPE search
    • Do not add new model families in this pass.

    Rescue Config Design

    Core mechanics

    • trainable.model_overrides.<model_type> lets each trainable family carry its own rescue optuna_config.
    • enqueue_trials stores explicit exact and neighbor seeds.
    • float_ranges and int_ranges allow split neighborhoods inside a single rescue experiment.

    Why enqueue_trials

    • Exact survivor replay is explicit.
    • Neighbor replay is explicit.
    • The seeded trials remain normal Optuna trials:
    • normal pruning
    • normal metrics
    • normal DB rows

    Why float_ranges and int_ranges

    • Some useful rescue areas are split.
    • The rescue stays as one experiment per model/fold instead of duplicating experiments only to express two islands of values.

    Model Rescue Logic

    mlp

    • Main survivor pattern retained:
    • hidden_dim=256
    • batch_size in {256, 512}
    • low or moderate dropout
    • cross_entropy common
    • silu strong
    • Rescue intent:
    • replay exact sibling survivors
    • explore a small neighborhood around them
    • avoid reopening the full Tier A space

    tcn

    • Main survivor pattern retained:
    • channels=32
    • activation=silu
    • kernel_size in {3, 5}
    • num_layers in {2, 3}
    • low learning_rate
    • very low weight_decay
    • dropout around 0.1-0.3
    • Rescue intent:
    • transfer surviving F3/F4 settings into F1/F2
    • test nearby variants without broad random expansion

    vit

    • Main survivor pattern retained:
    • batch high
    • cross_entropy
    • relative_bias-leaning positional settings
    • patch_size=4, patch_stride=2 around the strongest seeds
    • Rescue intent:
    • keep the model family
    • narrow the space
    • seed exact and nearby survivors

    axial_vit

    • AXV-F2 exact survivors used as the first family-specific seeds:
    • trial_id=1351
    • trial_id=1355
    • Shared retained pattern:
    • hidden_dim=96
    • time_num_heads=2
    • feature_num_heads=8
    • num_layers=6
    • dim_feedforward=512
    • activation=relu
    • norm_first=false
    • dropout=0.2
    • batch_size=512
    • patch_size=4
    • patch_stride=2
    • pooling_type_2d=attention
    • attn_pool_hidden_dim=128
    • attn_pool_dropout=0.2
    • low learning_rate
    • low weight_decay
    • Additional transfer rule:
    • shared ideas from vit are allowed only on compatible knobs
    • not as a fake one-to-one architecture copy

    Ranged Enqueue Fix

    • Warnings appeared when old generated rescue payloads enqueued fixed values for ranged params without the matching __range_index.
    • Source rescue JSON was checked after the fix:
    • 0 window_size seeds outside declared ranges
    • 0 learning_rate seeds outside declared ranges
    • 0 weight_decay seeds outside declared ranges
    • The warning source was stale generated payloads.
    • The canonical manifest set after that fix is: v20260323_tierA_rescue_v2

    .venv/bin/python scripts/thesis/script_2_thesis/generate_experiments/generate_script2_experiments.py \
  --config-path scripts/thesis/script_2_thesis/generate_experiments/script2_tiers_config_v2_tierA_rescue.json \
  --output-dir scripts/thesis/script_2_thesis/experiments \
  --version-tag v20260323_tierA_rescue_v2 \
  --skip-active-manifest

    rg "__range_index" scripts/thesis/script_2_thesis/experiments/v20260323_tierA_rescue_v2

    bash scripts/thesis/script_2_thesis/run_script2.sh \
