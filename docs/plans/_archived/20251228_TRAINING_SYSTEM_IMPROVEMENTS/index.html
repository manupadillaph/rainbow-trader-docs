    • Key Insight: Class weights must be coordinated with sampling ratio:

    • If sampling is aggressive (1:1 ratio) → data already balanced → weights should be moderate

    • If sampling is less aggressive (3:1 ratio) → data less balanced → weights can be more extreme

    • Cannot use extreme weights (0.1 for NONE) with aggressive sampling — would over-correct

    • Coordination Logic:

    ```python

    # Example: Calculate weights based on sampling config def calculate_class_weights(sampling_config: SamplingConfig) -> torch.Tensor: """Calculate class weights that complement (not duplicate) sampling effect."""

      # Base weights inversely proportional to sampling ratio
  # If none_ratio=1.0 (balanced) → NONE weight ≈ 0.8-1.0
  # If none_ratio=3.0 (less balanced) → NONE weight ≈ 0.5
  # If none_ratio=0.5 (very balanced) → NONE weight ≈ 1.0 (no adjustment needed)

  training_ratio = sampling_config.training_none_ratio

  # But never too extreme because sampling already handles it
  none_weight = max(0.5, 1.0 / (1.0 + training_ratio * 0.3))

  return torch.tensor([none_weight, 1.0, 1.0])  # [NONE, LONG, SHORT]
    ```
